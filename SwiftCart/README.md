## WELCOME TO ASSIGNMENT (SwiftCart E-Commerce) Questions and Answers.


##

#### 1) What is the difference between `null` and `undefined`?
    ✅Undefined: যখন কোনো ভ্যারিয়েবল ডিক্লেয়ার করা হয় কিন্তু কোনো ভ্যালু অ্যাসাইন করা হয় না, তখন জাভাস্ক্রিপ্ট ডিফল্টভাবে তাকে undefined বলে।
    Null: এটি একটি অ্যাসাইনমেন্ট ভ্যালু। যখন প্রোগ্রামার ইচ্ছাকৃতভাবে কোনো ভ্যারিয়েবলকে খালি বা "মান নেই" হিসেবে বোঝাতে চান, তখন null ব্যবহার করা হয়।

#### 2) What is the use of the `map()` function in JavaScript? How is it different from `forEach()`?
    map() একটি অ্যারের প্রতিটি উপাদানের ওপর কাজ করে এবং ফলাফল হিসেবে একটি নতুন অ্যারে রিটার্ন করে।
    forEach() শুধু অ্যারের প্রতিটি উপাদানের ওপর কাজ করে এবং কোনো ফলাফল রিটার্ন করে না।

#### 3) What is the difference between `==` and `===`?
    == (Loose Equality): এটি শুধু মান (value) চেক করে, ডাটা টাইপ চেক করে না। (যেমন: 5 == "5" সত্য)।
    === (Strict Equality): এটি মান এবং ডাটা টাইপ উভয়ই চেক করে। (যেমন: 5 === "5" মিথ্যা)।

#### 4) What is the significance of `async`/`await` in fetching API data?
    async/await: async/await জাভাস্ক্রিপ্টের অ্যাসিনক্রোনাস কাজগুলোকে আরও সহজ এবং রিডেবল করে তোলে। এটি প্রমিস (Promise) হ্যান্ডেল করার একটি আধুনিক উপায় যা কোডকে দেখতে সিঙ্ক্রোনাস কোডের মতো করে তোলে, ফলে এরর হ্যান্ডলিং সহজ হয়।

#### 5) Explain the concept of Scope in JavaScript (Global, Function, Block).
    Global Scope: ভ্যারিয়েবল যদি কোডের একদম শুরুতে ডিক্লেয়ার করা হয় এবং যেকোনো জায়গা থেকে অ্যাক্সেস করা যায়।
    Function Scope: ভ্যারিয়েবল যদি কোনো ফাংশনের ভেতর ডিক্লেয়ার করা হয়, তবে তা শুধু ওই ফাংশনের ভেতরেই কাজ করবে।
    Block Scope: let এবং const দিয়ে ডিক্লেয়ার করা ভ্যারিয়েবল শুধু নির্দিষ্ট সেকেন্ড ব্র্যাকেট { } (যেমন if বা for loop) এর ভেতর কাজ করে।
